# Quickstart: Internet Search RAG

## Prerequisites
1. Ensure GEMINI_API_KEY is set in .env or environment.
2. Install dependencies:
   `ash
   .\google-adk-venv\Scripts\pip.exe install duckduckgo-search
   `

## Running the Agent
`ash
.\google-adk-venv\Scripts\python.exe rag_internet_search.py
`

## Usage
1. **Select Agent**: Choose a number from the list (or press Enter for default).
2. **Ask Question**: Enter your query.
3. **Get Details**: If you want more than the 2-3 line answer, type "explain" or "more details".
